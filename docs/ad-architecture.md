# Active Directory Architecture

This document outlines the structure and configuration of the Active Directory environment created in this lab.

## Domain Information

- **Domain Name:** beshornlab.local  
- **Forest Functional Level:** Windows Server 2022  
- **DNS Installed:** Yes  
- **Domain Controller:** Windows Server 2022 EC2 instance  

## Organizational Units (OUs)

The following OUs were created to simulate a real company structure:

- **Human Resources**  
- **IT Department**
