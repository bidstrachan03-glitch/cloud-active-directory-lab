# Ticketing Workflow Documentation

This document outlines the IT Help Desk ticketing workflow used in this project to simulate real-world support tasks.

## Ticket #001 – New Employee Account Creation

**Summary:**  
HR requested a new Active Directory account for John Smith.

### Steps Completed

1. Received ticket request  
2. Created user account in the Human Resources OU  
3. Assigned username: jsmith  
4. Configured initial password  
5. Added user to the "Employees" security group  
6. Documented completion in the ticket  

### Status  
**Closed**

## Purpose of Ticketing in This Lab

Using tickets helps simulate:

- Real IT support workflows  
- Documentation practices  
- Task tracking  
- Communication between departments  

This mirrors what Help Desk and SysAdmin teams do daily.
