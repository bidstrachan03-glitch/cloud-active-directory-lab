# Project Overview

This project demonstrates the deployment and configuration of a cloud-based Active Directory environment using Amazon Web Services (AWS) and Windows Server 2022. The goal of this lab is to simulate real-world IT administration tasks, including domain setup, user management, security group creation, ticketing workflows, and Jira task tracking.

## Objectives

- Deploy a Windows Server EC2 instance on AWS  
- Install and configure Active Directory Domain Services (AD DS)  
- Create Organizational Units (OUs)  
- Add users and security groups  
- Document IT Help Desk tasks  
- Track work using Jira  
- Build a professional, portfolio-ready GitHub repository  

## Skills Demonstrated

- Cloud Administration (AWS EC2)  
- Windows Server Management  
- Active Directory Configuration  
- PowerShell Scripting  
- IT Support Ticketing  
- Jira Workflow Management  
- Technical Documentation
